## Installation

Install the project with Vs Code

```bash
 install vs code
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/erpushpendrasingh/cred-quantact
```

Open vs-code terminal

```bash
  Go inside project folder
```

```bash
 npm install
```

Run project

```bash
 npm start
```
